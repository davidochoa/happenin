### Database
* create db role 'happenin'
* run 'rake db:reset'
* run 'rake db:migrate'

* Should be able to start the server

### Environment variables
#### Environment variables are handled with [dotenv](https://github.com/bkeepers/dotenv) gem
- DB_DATABASE
- DB_HOST
- DB_USERNAME
- DB_PASSWORD
